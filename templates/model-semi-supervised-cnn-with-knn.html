There are two models running sequentially on this page on the MNIST dataset.
Take a look at the input data by clicking the Input Data tab.

The first model trains a CNN on 5,500 labelled training examples. You can
see the model architecture and live training output in the Model tab.
Once it finishes, you can see accuracy per class in the Evaluation tab.

The overall accuracy is about 89%. This is great, but to have 5,500 labelled
examples in real world applications would be expensive. We'll train the
same CNN model on only 50 labelled examples to compare.

You can see the live training in the Model-50 tab. The Evaluation for Model-50
tab will show you the poor results. The overall accuracy is only 25%. Still better
than random chance though.

Using semi-supervised learning with k-means clustering, we can achieve much higher
accuracy with only 50 trained examples. Since k-mean method is more computationally
expensive on the MNIST dataset for ~50,000 examples, it's best done in Colaboratory.
To see the machine do the work of labelling, follow the link below.
